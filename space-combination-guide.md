# Space Visualizer - Material Combination Creation Guide

## Quick Reference
- **Total Images to Create**: 256 (64 per space × 4 spaces)
- **Material Options**: 4 countertops × 4 backsplashes × 4 cabinets = 64 combinations
- **File Naming**: `space[1-4]_[countertop]_[backsplash]_[cabinet].jpg`

## QUICK CREATION WORKFLOW

### Batch Creation Strategy
For efficiency, consider creating images in this order:

1. **Choose one space** (e.g., Space 1)
2. **Start with one countertop** (e.g., African Rainbow)
3. **Create all 16 combinations** for that countertop
4. **Move to next countertop** and repeat
5. **Complete the space** then move to next space

### File Organization
Create this directory structure:
```
assets/
└── space-combinations/
    ├── space1/
    ├── space2/
    ├── space3/
    └── space4/
```
